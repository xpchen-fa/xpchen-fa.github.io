дЄАгАБJavaScriptжШѓеНХзЇњз®Л

JavaScriptзЪДеНХзЇњз®ЛпЉМдЄОеЃГзЪДзФ®йАФжЬЙеЕ≥гАВдљЬдЄЇжµПиІИеЩ®иДЪжЬђиѓ≠и®АпЉМJavaScriptзЪДдЄїи¶БзФ®йАФжШѓдЄОзФ®жИЈдЇТеК®пЉМдї•еПКжУНдљЬDOMгАВињЩеЖ≥еЃЪдЇЖеЃГеП™иГљжШѓеНХзЇњз®ЛпЉМеР¶еИЩдЉЪеЄ¶жЭ•еЊИе§НжЭВзЪДеРМж≠•йЧЃйҐШгАВжѓФе¶ВпЉМеБЗеЃЪJavaScriptеРМжЧґжЬЙдЄ§дЄ™зЇњз®ЛпЉМдЄАдЄ™зЇњз®ЛеЬ®жЯРдЄ™DOMиКВзВєдЄКжЈїеК†еЖЕеЃєпЉМеП¶дЄАдЄ™зЇњз®ЛеИ†йЩ§дЇЖињЩдЄ™иКВзВєпЉМињЩжЧґжµПиІИеЩ®еЇФиѓ•дї•еУ™дЄ™зЇњз®ЛдЄЇеЗЖпЉЯ

дЇМгАБдїїеК°йШЯеИЧ

еНХзЇњз®Ле∞±жДПеС≥зЭАпЉМжЙАжЬЙдїїеК°йЬАи¶БжОТйШЯпЉМеЙНдЄАдЄ™дїїеК°зїУжЭЯпЉМжЙНдЉЪжЙІи°МеРОдЄАдЄ™дїїеК°гАВе¶ВжЮЬеЙНдЄАдЄ™дїїеК°иАЧжЧґеЊИйХњпЉМеРОдЄАдЄ™дїїеК°е∞±дЄНеЊЧдЄНдЄАзЫіз≠ЙзЭАгАВ

е¶ВжЮЬжОТйШЯжШѓеЫ†дЄЇиЃ°зЃЧйЗПе§ІпЉМCPUењЩдЄНињЗжЭ•пЉМеАТдєЯзЃЧдЇЖпЉМдљЖжШѓеЊИе§ЪжЧґеАЩCPUжШѓйЧ≤зЭАзЪДпЉМеЫ†дЄЇIOиЃЊе§ЗпЉИиЊУеЕ•иЊУеЗЇиЃЊе§ЗпЉЙеЊИжЕҐпЉИжѓФе¶ВAjaxжУНдљЬдїОзљСзїЬиѓїеПЦжХ∞жНЃпЉЙпЉМдЄНеЊЧдЄНз≠ЙзЭАзїУжЮЬеЗЇжЭ•пЉМеЖНеЊАдЄЛжЙІи°МгАВ

JavaScriptиѓ≠и®АзЪДиЃЊиЃ°иАЕжДПиѓЖеИ∞пЉМињЩжЧґдЄїзЇњз®ЛеЃМеЕ®еПѓдї•дЄНзЃ°IOиЃЊе§ЗпЉМжМВиµЈе§ДдЇОз≠ЙеЊЕдЄ≠зЪДдїїеК°пЉМеЕИињРи°МжОТеЬ®еРОйЭҐзЪДдїїеК°гАВз≠ЙеИ∞IOиЃЊе§ЗињФеЫЮдЇЖзїУжЮЬпЉМеЖНеЫЮињЗе§іпЉМжККжМВиµЈзЪДдїїеК°зїІзї≠жЙІи°МдЄЛеОїгАВ

дЇОжШѓпЉМжЙАжЬЙдїїеК°еПѓдї•еИЖжИРдЄ§зІНпЉМдЄАзІНжШѓеРМж≠•дїїеК°пЉИsynchronousпЉЙпЉМеП¶дЄАзІНжШѓеЉВж≠•дїїеК°пЉИasynchronousпЉЙгАВеРМж≠•дїїеК°жМЗзЪДжШѓпЉМеЬ®дЄїзЇњз®ЛдЄКжОТйШЯжЙІи°МзЪДдїїеК°пЉМеП™жЬЙеЙНдЄАдЄ™дїїеК°жЙІи°МеЃМжѓХпЉМжЙНиГљжЙІи°МеРОдЄАдЄ™дїїеК°пЉЫеЉВж≠•дїїеК°жМЗзЪДжШѓпЉМдЄНињЫеЕ•дЄїзЇњз®ЛгАБиАМињЫеЕ•вАЭдїїеК°йШЯеИЧвАЭпЉИtask queueпЉЙзЪДдїїеК°пЉМеП™жЬЙвАЭдїїеК°йШЯеИЧвАЭйАЪзЯ•дЄїзЇњз®ЛпЉМжЯРдЄ™еЉВж≠•дїїеК°еПѓдї•жЙІи°МдЇЖпЉМиѓ•дїїеК°жЙНдЉЪињЫеЕ•дЄїзЇњз®ЛжЙІи°МгАВ

дЄЙгАБдЇЛдїґеТМеЫЮи∞ГеЗљжХ∞

вАЬдїїеК°йШЯеИЧвАЭжШѓдЄАдЄ™дЇЛдїґзЪДйШЯеИЧпЉИдєЯеПѓдї•зРЖиІ£жИРжґИжБѓзЪДйШЯеИЧпЉЙпЉМIOиЃЊе§ЗеЃМжИРдЄАй°єдїїеК°пЉМе∞±еЬ®вАЭдїїеК°йШЯеИЧвАЭдЄ≠жЈїеК†дЄАдЄ™дЇЛдїґпЉМи°®з§ЇзЫЄеЕ≥зЪДеЉВж≠•дїїеК°еПѓдї•ињЫеЕ•вАЭжЙІи°Мж†ИвАЭдЇЖгАВдЄїзЇњз®ЛиѓїеПЦвАЭдїїеК°йШЯеИЧвАЭпЉМе∞±жШѓиѓїеПЦйЗМйЭҐжЬЙеУ™дЇЫдЇЛдїґгАВ

вАЬдїїеК°йШЯеИЧвАЭдЄ≠зЪДдЇЛдїґпЉМйЩ§дЇЖIOиЃЊе§ЗзЪДдЇЛдїґдї•е§ЦпЉМињШеМЕжЛђдЄАдЇЫзФ®жИЈдЇІзФЯзЪДдЇЛдїґпЉИжѓФе¶ВйЉ†ж†ЗзВєеЗїгАБй°µйЭҐжїЪеК®з≠Йз≠ЙпЉЙгАВеП™и¶БжМЗеЃЪињЗеЫЮи∞ГеЗљжХ∞пЉМињЩдЇЫдЇЛдїґеПСзФЯжЧґе∞±дЉЪињЫеЕ•вАЭдїїеК°йШЯеИЧвАЭпЉМз≠ЙеЊЕдЄїзЇњз®ЛиѓїеПЦгАВ

жЙАи∞УвАЭеЫЮи∞ГеЗљжХ∞вАЭпЉИcallbackпЉЙпЉМе∞±жШѓйВ£дЇЫдЉЪиҐЂдЄїзЇњз®ЛжМВиµЈжЭ•зЪДдї£з†БгАВеЉВж≠•дїїеК°ењЕй°їжМЗеЃЪеЫЮи∞ГеЗљжХ∞пЉМељУдЄїзЇњз®ЛеЉАеІЛжЙІи°МеЉВж≠•дїїеК°пЉМе∞±жШѓжЙІи°МеѓєеЇФзЪДеЫЮи∞ГеЗљжХ∞гАВ

вАЬдїїеК°йШЯеИЧвАЭжШѓдЄАдЄ™еЕИињЫеЕИеЗЇзЪДжХ∞жНЃзїУжЮДпЉМжОТеЬ®еЙНйЭҐзЪДдЇЛдїґпЉМдЉШеЕИиҐЂдЄїзЇњз®ЛиѓїеПЦгАВдЄїзЇњз®ЛзЪДиѓїеПЦињЗз®ЛеЯЇжЬђдЄКжШѓиЗ™еК®зЪДпЉМеП™и¶БжЙІи°Мж†ИдЄАжЄЕз©ЇпЉМвАЭдїїеК°йШЯеИЧвАЭдЄКзђђдЄАдљНзЪДдЇЛдїґе∞±иЗ™еК®ињЫеЕ•дЄїзЇњз®ЛгАВдљЖжШѓпЉМзФ±дЇОе≠ШеЬ®еРОжЦЗжПРеИ∞зЪДвАЭеЃЪжЧґеЩ®вАЭеКЯиГљпЉМдЄїзЇњз®Лй¶ЦеЕИи¶Бж£АжЯ•дЄАдЄЛжЙІи°МжЧґйЧіпЉМжЯРдЇЫдЇЛдїґеП™жЬЙеИ∞дЇЖиІДеЃЪзЪДжЧґйЧіпЉМжЙНиГљињФеЫЮдЄїзЇњз®ЛгАВ

еЫЫгАБEvent Loop

ињРи°МжЬЇеИґпЉЪ

  1. жЙАжЬЙеРМж≠•дїїеК°йГљеЬ®дЄїзЇњз®ЛдЄКжЙІи°МпЉМељҐжИРдЄАдЄ™жЙІи°Мж†И(execution context stackпЉЙгАВ
  2. дЄїзЇњз®ЛдєЛе§ЦпЉМињШе≠ШеЬ®дЄАдЄ™вАЭtask queueвАЭгАВеП™и¶БеЉВж≠•дїїеК°жЬЙдЇЖињРи°МзїУжЮЬпЉМе∞±еЬ®вАЭtask queueвАЭдєЛдЄ≠жФЊзљЃдЄАдЄ™дЇЛдїґгАВ
  3. дЄАжЧ¶вАЭжЙІи°Мж†ИвАЭдЄ≠зЪДжЙАжЬЙеРМж≠•дїїеК°жЙІи°МеЃМжѓХпЉМз≥їзїЯе∞±дЉЪиѓїеПЦвАЭtask queueвАЭпЉМзЬЛзЬЛйЗМйЭҐжЬЙеУ™дЇЫдЇЛдїґгАВйВ£дЇЫеѓєеЇФзЪДеЉВж≠•дїїеК°пЉМдЇОжШѓзїУжЭЯз≠ЙеЊЕзКґжАБпЉМињЫеЕ•жЙІи°Мж†ИпЉМеЉАеІЛжЙІи°МгАВ
  4. дЄїзЇњз®ЛдЄНжЦ≠йЗНе§НдЄКйЭҐзЪДзђђдЄЙж≠•гАВ

дЇФгАБеЃЪжЧґеЩ®

йЩ§дЇЖжФЊзљЃеЉВж≠•дїїеК°зЪДдЇЛдїґпЉМвАЭдїїеК°йШЯеИЧвАЭињШеПѓдї•жФЊзљЃеЃЪжЧґдЇЛдїґпЉМеН≥жМЗеЃЪжЯРдЇЫдї£з†БеЬ®е§Ъе∞СжЧґйЧідєЛеРОжЙІи°МгАВињЩеПЂеБЪвАЭеЃЪжЧґеЩ®вАЭпЉИtimerпЉЙеКЯиГљпЉМдєЯе∞±жШѓеЃЪжЧґжЙІи°МзЪДдї£з†БгАВ

еЃЪжЧґеЩ®еКЯиГљдЄїи¶БзФ±setTimeout()еТМsetInterval()ињЩдЄ§дЄ™еЗљжХ∞жЭ•еЃМжИРпЉМеЃГдїђзЪДеЖЕйГ®ињРи°МжЬЇеИґеЃМеЕ®дЄАж†ЈпЉМеМЇеИЂеЬ®дЇОеЙНиАЕжМЗеЃЪзЪДдї£з†БжШѓдЄАжђ°жАІжЙІи°МпЉМеРОиАЕеИЩдЄЇеПНе§НжЙІи°МгАВ

HTML5ж†ЗеЗЖиІДеЃЪдЇЖsetTimeout()зЪДзђђдЇМдЄ™еПВжХ∞зЪДжЬАе∞ПеАЉпЉИжЬАзЯ≠йЧійЪФпЉЙпЉМдЄНеЊЧдљОдЇО4жѓЂзІТпЉМе¶ВжЮЬдљОдЇОињЩдЄ™еАЉпЉМе∞±дЉЪиЗ™еК®еҐЮеК†гАВеЬ®ж≠§дєЛеЙНпЉМиАБзЙИжЬђзЪДжµПиІИеЩ®йГље∞ЖжЬАзЯ≠йЧійЪФиЃЊдЄЇ10жѓЂзІТгАВеП¶е§ЦпЉМеѓєдЇОйВ£дЇЫDOMзЪДеПШеК®пЉИе∞§еЕґжШѓжґЙеПКй°µйЭҐйЗНжЦ∞жЄ≤жЯУзЪДйГ®еИЖпЉЙпЉМйАЪеЄЄдЄНдЉЪзЂЛеН≥жЙІи°МпЉМиАМжШѓжѓП16жѓЂзІТжЙІи°МдЄАжђ°гАВињЩжЧґдљњзФ®requestAnimationFrame()зЪДжХИжЮЬи¶Бе•љдЇОsetTimeout()гАВ

йЬАи¶Бж≥®жДПзЪДжШѓпЉМsetTimeout()еП™жШѓе∞ЖдЇЛдїґжПТеЕ•дЇЖвАЭдїїеК°йШЯеИЧвАЭпЉМењЕй°їз≠ЙеИ∞ељУеЙНдї£з†БпЉИжЙІи°Мж†ИпЉЙжЙІи°МеЃМпЉМдЄїзЇњз®ЛжЙНдЉЪеОїжЙІи°МеЃГжМЗеЃЪзЪДеЫЮи∞ГеЗљжХ∞гАВи¶БжШѓељУеЙНдї£з†БиАЧжЧґеЊИйХњпЉМжЬЙеПѓиГљи¶Бз≠ЙеЊИдєЕпЉМжЙАдї•еєґж≤°жЬЙеКЮж≥ХдњЭиѓБпЉМеЫЮи∞ГеЗљжХ∞дЄАеЃЪдЉЪеЬ®setTimeout()жМЗеЃЪзЪДжЧґйЧіжЙІи°МгАВ

еЕ≠гАБNode.jsзЪДEvent Loop

Node.jsдєЯжШѓеНХзЇњз®ЛзЪДEvent LoopпЉМдљЖжШѓеЃГзЪДињРи°МжЬЇеИґдЄНеРМдЇОжµПиІИеЩ®зОѓеҐГгАВ

ињРи°МжЬЇеИґпЉЪ

  1. V8еЉХжУОиІ£жЮРJavaScriptиДЪжЬђгАВ
  2. иІ£жЮРеРОзЪДдї£з†БпЉМи∞ГзФ®Node APIгАВ
  3. libuvеЇУиіЯиі£Node APIзЪДжЙІи°МгАВеЃГе∞ЖдЄНеРМзЪДдїїеК°еИЖйЕНзїЩдЄНеРМзЪДзЇњз®ЛпЉМељҐжИРдЄАдЄ™Event LoopпЉИдЇЛдїґеЊ™зОѓпЉЙпЉМдї•еЉВж≠•зЪДжЦєеЉПе∞ЖдїїеК°зЪДжЙІи°МзїУжЮЬињФеЫЮзїЩV8еЉХжУОгАВ
  4. V8еЉХжУОеЖНе∞ЖзїУжЮЬињФеЫЮзїЩзФ®жИЈгАВ

йЩ§дЇЖsetTimeoutеТМsetIntervalињЩдЄ§дЄ™жЦєж≥ХпЉМNode.jsињШжПРдЊЫдЇЖеП¶е§ЦдЄ§дЄ™дЄОвАЭдїїеК°йШЯеИЧвАЭжЬЙеЕ≥зЪДжЦєж≥ХпЉЪprocess.nextTickеТМsetImmediateгАВ

process.nextTickжЦєж≥ХеПѓдї•еЬ®ељУеЙНвАЭжЙІи°Мж†ИвАЭзЪДе∞ЊйГ®вАФ-дЄЛдЄАжђ°Event LoopпЉИдЄїзЇњз®ЛиѓїеПЦвАЭдїїеК°йШЯеИЧвАЭпЉЙдєЛеЙНвАФ-иІ¶еПСеЫЮи∞ГеЗљжХ∞гАВдєЯе∞±жШѓиѓіпЉМеЃГжМЗеЃЪзЪДдїїеК°жАїжШѓеПСзФЯеЬ®жЙАжЬЙеЉВж≠•дїїеК°дєЛеЙНгАВsetImmediateжЦєж≥ХеИЩжШѓеЬ®ељУеЙНвАЭдїїеК°йШЯеИЧвАЭзЪДе∞ЊйГ®жЈїеК†дЇЛдїґпЉМдєЯе∞±жШѓиѓіпЉМеЃГжМЗеЃЪзЪДдїїеК°жАїжШѓеЬ®дЄЛдЄАжђ°Event LoopжЧґжЙІи°МпЉМињЩдЄОsetTimeout(fn, 0)еЊИеГПгАВ

дЄКйЭҐдї£з†БдЄ≠пЉМзФ±дЇОprocess.nextTickжЦєж≥ХжМЗеЃЪзЪДеЫЮи∞ГеЗљжХ∞пЉМжАїжШѓеЬ®ељУеЙНвАЭжЙІи°Мж†ИвАЭзЪДе∞ЊйГ®иІ¶еПСпЉМжЙАдї•дЄНдїЕеЗљжХ∞AжѓФsetTimeoutжМЗеЃЪзЪДеЫЮи∞ГеЗљжХ∞timeoutеЕИжЙІи°МпЉМиАМдЄФеЗљжХ∞BдєЯжѓФtimeoutеЕИжЙІи°МгАВињЩиѓіжШОпЉМе¶ВжЮЬжЬЙе§ЪдЄ™process.nextTickиѓ≠еП•пЉИдЄНзЃ°еЃГдїђжШѓеР¶еµМе•ЧпЉЙпЉМе∞ЖеЕ®йГ®еЬ®ељУеЙНвАЭжЙІи°Мж†ИвАЭжЙІи°МгАВ

зО∞еЬ®пЉМеЖНзЬЛsetImmediateгАВ
