最后

二、终端美化

1. 下载安装iterm2

官网：https://www.iterm2.com

在官网下载安装包，将安装包移到应用程序，然后打开就可以使用。

2. 更新zsh并切换shell为zsh

macOS 一般会自带 zsh，不过版本会比较早，我们先更新一下，以便使用最新特性。

由于我的电脑shell本来就是子啊zsh的所以没有进行这一步

3. 安装 oh-my-zsh

4. 安装额外的字体库

某些主题会用到一些其他字体，所以安装Powerline字体: https://github.com/powerline/fonts

安装好之后，选择一款Powerline字体了：iterm2 -> Preferences -> Profiles -> Text -> Font -> Change Font（我用的是Meslo LG）

5. 安装配色方案

iTerm2支持许多的主题配色，可以自己定义，也可以参考网上现成的主题配色。
iTerm2 配色合集GitHub地址：https://github.com/mbadolato/iTerm2-Color-Schemes/tree/master/schemes
我选择的是Nord配色方案 地址：https://github.com/arcticicestudio/nord-iterm2/releases/tag/v0.2.0
我是直接下载下来解压后然后直接导入到iTerm2中。
preferences → profiles → colors → color preset → import，选择下载的主题文件。

6. 修改主题配置。

首先打开家目录下的.zshrc文件

默认安装过后，使用的是robbyrussell主题，可以修改下面配置。

这个可以设置终端主题。我选择了这个agnoster，大家可以试一试。每次修改主题配置，都需要重启terminal终端。
其他主题可以参考Oh My Zsh主题展示网站
如果是其他定制主题，可以把主题文件下载到 ~/.oh-my-zsh/themes 里面，然后修改 ~/.zshrc 文件，配置好主题名字即可。

9. Zsh启动特技

可以通过一些配置，让Zsh在启动的时候输出启动语。

参考(这里)[https://xiaozhou.net/learn-the-command-line-iterm-and-zsh-2017-06-23.html]

10. iTerm背景特技

iTerm还支持显示背景图，这样我们还可以把单调的终端界面配置得更加酷炫。
配置方法：Preferences -> Profiles -> Window -> Background Image 选择一个你喜欢的背景，然后通过Blending调整图片背景的透明度。

四、高效插件

snipaste

(Snipaste)[https://www.snipaste.com/#] is a simple but powerful snipping tool, and also allows you to pin the screenshot back onto the screen

Bob

(Bob)[https://github.com/ripperhe/Bob] 是一款 免费开源的 Mac 端翻译软件，支持划词翻译和截图翻译，翻译引擎支持有道翻译、百度翻译和谷歌翻译，手动输入进行翻译也是可以的。
(快速开始)[https://ripperhe.gitee.io/bob/#/general/quickstart/permission?id=%e5%bc%80%e5%90%af-chrome-%e7%9b%b8%e5%85%b3%e6%9d%83%e9%99%90]
